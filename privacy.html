Verdia – Privacy Policy

Last updated: February 2026

1. Data Controller

Verdia is a personal project operated from Spain by an independent developer.

Contact email: studioverdia@gmail.com

2. Data We Collect

Account Information

  • Email address
  • Optional username or alias
  • Optional profile photo
  • Unique user ID (generated via Supabase)

User Content

  • Photos of plants uploaded for analysis

Technical & Usage Data

  • IP address (logged for security purposes via Supabase)
  • Device information (Android OS version)
  • Usage metrics (number of scans, token consumption for cost management)
  • Notification preferences

Payment Information

Subscriptions are processed through Google Play Billing and RevenueCat. Verdia does not access or store credit card information.

3. Purpose of Processing

Data is processed to:

  • Provide AI-powered plant diagnosis
  • Manage subscriptions
  • Ensure platform security
  • Send irrigation reminders
  • Improve system performance

4. Third-Party Services

Verdia uses the following service providers:

  • Supabase (database and authentication services)
  • AWS infrastructure (cloud hosting)
  • Google Gemini API (AI plant analysis)
  • RevenueCat (subscription management)
  • Google Play Billing (payment processing)

5. International Data Transfers

Some service providers may process data outside the European Economic Area. Transfers are subject to appropriate safeguards.

6. Data Retention

Data is retained while the account remains active. Users may delete their account at any time from the app settings, which permanently removes associated personal data.

7. Legal Basis (GDPR)

  • User consent
  • Performance of a requested service
  • Legitimate interest in maintaining platform security

8. User Rights

Users may request access, rectification, deletion, or data portability by contacting:

studioverdia@gmail.com

9. Security Measures

Reasonable technical and organizational measures are implemented to protect data against unauthorized access or loss.
